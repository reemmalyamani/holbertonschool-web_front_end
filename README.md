# HTML Advanced

## Description
This project focuses on building the structure of a webpage using **pure HTML5**.  
The goal is to understand semantic HTML, proper page structure, and how to organize content before applying any styling with CSS.

All files are part of the **Holberton School Web Front-End curriculum**.

---

## Learning Objectives

By completing this project, I learned how to:

- Structure a webpage using **HTML5**
- Use semantic tags such as:
  - `header`
  - `main`
  - `section`
  - `article`
  - `aside`
  - `footer`
- Properly organize headings (`h1`, `h2`, `h3`)
- Build reusable page layouts
- Create accessible and well-structured HTML documents

---

## Requirements

- All files are written in **HTML5**
- No CSS or JavaScript was used
- Code is structured to follow **Holberton School guidelines**
